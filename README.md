# Lucas de Almeida - Portfolio

Personal portfolio of Lucas de Almeida, blockchain developer specializing in Web3, smart contracts, and DeFi.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How to run locally

```sh
# Install dependencies
npm install

# Run in development mode
npm run dev

# Build for production
npm run build
```

## Project structure

- `src/components/` - Reusable React components
- `src/pages/` - Application pages
- `src/hooks/` - Custom hooks
- `src/lib/` - Utilities and configurations
- `public/` - Static files
