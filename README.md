Lucas de Almeida - Personal Portfolio

Description
This repository contains the source code for my personal portfolio website, showcasing my work as a blockchain developer and Web3 enthusiast. The site highlights my expertise in smart contract development, DeFi protocols, security auditing, and Web3 infrastructure. It includes sections on my background, tech stack, featured projects, education, and contact information.
The portfolio is built with modern web technologies and is designed to be responsive, fast, and user-friendly. It serves as a professional showcase to attract collaborations and opportunities in the blockchain and DeFi space.
Live demo: lucasalmeida.me

Technologies Used

Framework: React with Vite for fast development and building.
Language: TypeScript for type-safe code.
Styling: Tailwind CSS for utility-first styling, with shadcn-ui for components.
Linting & Formatting: ESLint and PostCSS.
Other: Custom hooks, utilities, and static assets in public/.

Project Structure

src/components/: Reusable React components (e.g., headers, cards for projects).
src/pages/: Main application pages (e.g., home/about/projects).
src/hooks/: Custom React hooks for state management or utilities.
src/lib/: Shared utilities, configurations, and helpers.
public/: Static files like images, fonts, or icons.
Configuration files: tailwind.config.ts, vite.config.ts, tsconfig.json, etc.
